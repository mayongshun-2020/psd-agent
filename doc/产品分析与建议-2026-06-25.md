# 二、最大的问题

真正的问题有三个。

* * *

## 问题1：没有设计评分系统

这是最大的缺失。

目前PRD：

```
生成→ 导出
```

结束了。

但现实中：

品牌方一定会问：

> 为什么这个页面是好的？

所以必须增加：

# Design Score Agent

输出：

JSON

```
{  "brand_match":92,  "layout_quality":89,  "visual_consistency":95,  "readability":88,  "conversion_score":86}
```

最终：

```
综合评分：90.2
```

否则企业不敢用。

* * *

## 问题2：缺少人工反馈闭环

现在链路：

```
生成↓结束
```

实际需要：

```
生成↓设计师修改↓记录修改↓学习修改↓规则优化
```

否则：

AI永远不会越来越像品牌设计师。

建议增加：

```
Feedback Agent
```

记录：

-   删除了哪些模块
    
-   调整了哪些字体
    
-   修改了哪些文案
    

形成：

```
Human Preference Dataset
```

这是未来护城河。

* * *

## 问题3：没有设计资产生产系统

目前：

```
图片生成Agent
```

太简单。

实际上电商设计最大成本不是排版。

而是：

```
场景图卖点图功能图
```

建议拆成：

```
Image Studio Agent
```

下面再拆：

```
商品抠图Agent商品补光Agent商品重建Agent场景生成Agent卖点图生成Agent
```

否则最终效果不稳定。
