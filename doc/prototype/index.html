BrandOS AI Prototype

BrandOS AI 电商设计平台 HTML 原型

这套静态原型基于 `BRANDOS-AI-PRD.md` 和低保真文案整理，覆盖 MVP 主链路： 品牌训练、商品录入、设计任务创建、任务追踪，以及合并在任务详情中的结果预览与基础编辑。

8 张页面
覆盖核心业务端
静态 HTML
可直接本地打开
统一样式
便于后续转高保真
主链路
工作台 -> 品牌资产 -> 品牌规则 -> 商品管理 -> 创建设计任务 -> 设计任务 -> 任务详情（含结果编辑）

工作台

查看品牌概览、资产统计、最近训练任务和最近设计任务，并从快捷入口发起关键操作。

打开页面

品牌资产

作为品牌训练的输入中心，展示资产分类、列表、上传入口与训练状态。

打开页面

品牌规则

展示训练产出的 `design.md`、布局规则、组件摘要和 Prompt 模板。

打开页面

商品管理

录入商品资料、上传商品素材、维护卖点与 Brief，为任务生成提供结构化输入。

打开页面

创建设计任务

使用四步流程配置品牌、商品、生成参数和输出格式后提交异步任务。

打开页面

设计任务列表

统一查看任务状态、失败原因和结果入口，承接任务中心视角。

打开页面

任务详情

查看商品理解、页面结构、图片生成、布局结果，以及合并后的结果预览与基础编辑。

打开页面
